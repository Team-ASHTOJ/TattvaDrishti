# Docs Overview (docs/)

Purpose
- Centralized documentation for integrations, architecture, and deployment guides.

Current documents
- architecture.md: system-level architecture and data flow.
- FEDERATED_BLOCKCHAIN.md: ledger design, validation rules, and sync behavior.
- OLLAMA_SETUP.md / OLLAMA_INTEGRATION.md: local LLM setup and usage notes.
- project_summary.md: high-level project summary and roadmap ideas.

Support artifacts
- submission_logs_schema.sql / submission_logs_roles.sql: database schema ideas.
- insert_submission_log.py: helper for populating sample submissions.

Design intent
- Keep operational knowledge in the repo for quick onboarding.
